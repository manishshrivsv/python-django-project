🚗 Full Car Auction System

A real-time car auction web application that allows users to list vehicles, participate in live bidding, and manage auctions efficiently. The platform ensures a seamless and secure experience with dynamic bidding and real-time updates.

📌 Features
Real-Time Bidding
Implemented using WebSockets for instant bid updates without page refresh
Secure Authentication
User registration and login with validation and session management
Car Listings
Add, view, and manage car details with images and descriptions
Live Countdown Timer
Auctions are time-bound with automatic closure
Scalable Backend
Efficient data handling and optimized user flow
Responsive UI
Clean and user-friendly interface
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Python (Django)
Database: SQLite / MySQL
Real-Time: WebSockets
